# Android E-Commerce App Group Project

A simple yet functional E-Commerce Android application built for a Mobile Programming course project.
This app allows users to browse items, manage a shopping cart, view their order history, and update shipping details — all within a clean, modern UI.

## Features
### User Features

- Login & Logout – Secure user authentication system.

- View Account Information – Displays basic user details and profile picture.

- Change Shipping Address – Update your preferred delivery location.

- Order History – View past orders with item details and prices.

### Shopping Features

- Browse Products – Simple interface for viewing available items.

- Add to Cart / Remove from Cart – Manage your current shopping list easily.

- Cart Summary – Shows selected items, quantity, and total price.

- Checkout – Simulate a purchase and store it in the user’s order history.

## Tech Stack
- Component	Technology
- Language	Java

- UI Design	XML Layouts + ConstraintLayout
- Database	Firebase / SQLite (depending on your implementation)

- Version Control	Git & GitHub

## How It Works

Login: The user logs in using saved credentials.

Browse Items: The main screen lists available products.

Add to Cart: Each item can be added to the cart with quantity tracking.

Checkout: Confirms the order and stores it in the order history database.

Order History: Displays a RecyclerView of past purchases.

Uses M-Pesa Daraja Integration to Simulate Payments

## UI Highlights

Designed using ConstraintLayout for responsive layouts.

Custom rounded buttons and font styles for a clean look.

Uses RecyclerView for displaying dynamic lists (cart items, order history).

## How to Run

Clone the repository

Open in Android Studio

Go to File → Open → Select the project folder

Sync Gradle

Let Android Studio automatically sync dependencies.

Run on Emulator or Device

Connect an Android phone or use an emulator (API 30+ recommended).

Login and Explore!
